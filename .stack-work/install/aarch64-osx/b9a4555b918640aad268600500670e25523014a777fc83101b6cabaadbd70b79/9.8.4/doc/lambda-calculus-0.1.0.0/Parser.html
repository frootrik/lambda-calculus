Description

This module defines parsers for lambda expressions, including variables, abstractions, and applications. The resulting expressions are represented using the Lambda AST defined in LambdaAST.

parseLambda :: Parser (Lambda String) Source #

Parses any lambda calculus expression, trying abstraction, application, parenthesized expressions, or variables in order.

Returns a raw 'Lambda String' term (unscoped).

topParser :: String -> Either (ParseErrorBundle String Void) (Lambda String) Source #

Top-level parser that consumes the entire input string.

It parses a lambda expression and ensures no remaining input is left. Returns either a parse error or the parsed 'Lambda String' expression.
